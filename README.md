# SQLite Database Viewer (Browser-Based)

A **modern, browser-based SQLite database viewer** built with **HTML, CSS, and SQL.js**.  
No backend. No installs. Just open the file and explore your `.sqlite` / `.db` files securely in your browser.

---

## Features

- Open SQLite `.db` / `.sqlite` files directly in the browser
- **Dark mode** (default) with light mode toggle
- **Refresh database** without re-selecting the file
- View all tables automatically
- Display up to 50 rows per table
- Runs fully client-side (your data never leaves your machine)

---

## Demo

Just open the HTML file in your browser and select a SQLite database file.

No server required.

---

## Tech Stack

- **HTML5**
- **CSS3 (Custom Dark / Light Theme)**
- **JavaScript (Vanilla)**
- **[SQL.js](https://github.com/sql-js/sql.js)** (SQLite compiled to WebAssembly)
